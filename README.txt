A Simple Plugin to change WooCommerce SKU Label to ISBN or any other custom text. No need to touch any code or core WooCommerce functions.

== Description ==

**SKU Label Changer Now Support [Dokan â€“ Powerful WooCommerce Multivendor Marketplace Solution](https://wordpress.org/plugins/dokan-lite/)**

SKU Label Changer For WooCommerce is a simple but effective plugin for WooCommerce users. Most of the users wants to change the default SKU label. But, unfortunately they don't understand the codes. Also, hiring a web developer is not a good idea for change the simple text.

This plugin allows to to change the text easily. Here are the steps that you need to done.

- Install the plugin from WordPress.org Repository.
- After install you will find an option named 'SKU Changer' from WordPress sidebar. Click on it.
- Select Enable SKU
- Change the text from SKU Label textbox. Example: ISBN
- Click on 'Save Settings'
- Done
